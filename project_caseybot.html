CaseyBot - Salesforce AI Case Matcher

CaseyBot is an AI-powered chatbot integrated into Salesforce to match old cases with new ones, enabling faster and more efficient case resolution.

How It Was Built

The bot was built using Python to interface with the Salesforce API. Natural Language Processing (NLP) models were integrated to identify patterns in case descriptions and suggest relevant resolutions.

Languages and Tools

  • Python
  • Salesforce API
  • Natural Language Processing (NLP)
