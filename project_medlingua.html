MedLingua - AI Assisted Medical Scribe

MedLingua is an AI-powered web application that assists doctors and patients by translating complex medical terminology into everyday language, improving communication and patient understanding.

How It Was Built

The app was developed using Python for the AI models and SvelteKit for the frontend. It uses natural language processing (NLP) to convert medical notes into simpler language that is easier to understand for patients.

Languages and Tools

  • Python
  • SvelteKit
  • NLP

Code Example

                
from transformers import pipeline

# Load pre-trained model for medical text translation
translator = pipeline("translation", model="facebook/wmt19-en-de")

text = "The patient has a history of chronic bronchitis."
result = translator(text)
print(result)
