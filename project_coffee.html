Coffee Recommender - Personalized Coffee Suggestions

This project is a JavaScript-based web application that recommends coffee types based on user preferences. The app uses a simple quiz format to gather user inputs and suggests a coffee type based on their answers.

How It Was Built

The Coffee Recommender was built using JavaScript for interactivity and HTML/CSS for the user interface. It uses conditional logic to determine the best coffee match based on user responses.

Languages and Tools

  • JavaScript
  • HTML/CSS

Code Example

                
function getCoffeeRecommendation(strength, milk, sweetness) {
    if (strength === "strong" && milk === "none") {
        return "Espresso";
    } else if (strength === "medium" && sweetness === "yes") {
        return "Latte";
    }
    return "Americano";
}
