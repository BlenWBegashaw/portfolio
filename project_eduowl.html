EduOwl - University Admissions and Major Selection Assistant

EduOwl is a web application designed to help students with university admissions and selecting the right major. It uses an AI-driven recommendation system to analyze student preferences and suggest suitable majors.

How It Was Built

The project was built using JavaScript for the front end and integrated with AI models to evaluate student interests and recommend relevant majors. Web scraping was used to gather admission information from various sources.

Languages and Tools

  • JavaScript
  • Web Scraping
  • AI Models

Code Example

                
fetch("https://university-api.com/admissions")
    .then(response => response.json())
    .then(data => {
        // Process admission data and show on UI
        displayAdmissionDetails(data);
    });
